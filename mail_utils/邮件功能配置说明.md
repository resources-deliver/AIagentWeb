# 邮件功能配置说明

## 功能概述

系统已集成自动化邮件发送功能，支持通过自然语言指令自动生成和发送邮件。

## 配置步骤

### 1. 配置SMTP服务

复制`.env.example`为`.env`，并配置以下参数：

```bash
# SMTP服务器地址
SMTP_SERVER=smtp.qq.com

# SMTP端口
SMTP_PORT=587

# SMTP登录用户名
SMTP_USERNAME=your_email@qq.com

# SMTP密码（授权码）
SMTP_PASSWORD=your_authorization_code

# 发件人邮箱
FROM_EMAIL=your_email@qq.com

# 发件人显示名称
FROM_NAME=AI数字人助手
```

### 2. 获取SMTP授权码

#### QQ邮箱
1. 登录QQ邮箱网页版
2. 进入"设置" -> "账户"
3. 找到"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
4. 开启"IMAP/SMTP服务"
5. 生成授权码（16位字符）

#### Gmail
1. 登录Google账户
2. 进入"安全性"设置
3. 启用"两步验证"
4. 生成"应用专用密码"

#### 163邮箱
1. 登录163邮箱网页版
2. 进入"设置" -> "POP3/SMTP/IMAP"
3. 开启"IMAP/SMTP服务"
4. 设置授权密码

## 使用示例

### 基础邮件发送
```
用户：帮我写一封邮件，主题为"讨论毕业设计选题"，和老师预约下周合适的时间来讨论毕业设计的选题，发送给指导老师，邮箱为283406@qq.com
```

系统会自动：
- 提取主题："讨论毕业设计选题"
- 生成邮件正文：包含预约下周讨论毕业设计选题的内容
- 确定收件人：283406@qq.com
- 发送邮件

### 简单邮件
```
用户：给test@example.com发一封邮件，主题是"会议提醒"，内容是"明天下午3点开会"
```

### 商务邮件
```
用户：给client@company.com发商务邮件，主题是"项目合作意向"，内容是"我们公司对贵公司的项目很感兴趣，希望能有机会合作"
```

## 错误处理

系统会自动处理以下错误情况：

1. **无效邮箱格式**：提示用户邮箱地址不正确
2. **邮件服务未配置**：提示联系管理员配置SMTP服务
3. **认证失败**：提示检查SMTP账号密码
4. **发送失败**：显示具体错误信息

## 安全特性

- 邮箱地址格式验证
- SMTP TLS加密传输
- 敏感信息通过环境变量配置
- 详细的错误日志记录

## 技术实现

邮件功能通过Function Calling机制实现：

1. 用户发送自然语言指令
2. 大模型分析意图，提取邮件信息
3. 生成工具调用指令（包含收件人、主题、正文）
4. Agent执行send_email函数
5. 通过SMTP协议发送邮件
6. 返回发送状态给用户

## 支持的邮件服务商

- QQ邮箱 (smtp.qq.com)
- Gmail (smtp.gmail.com)
- 163邮箱 (smtp.163.com)
- 126邮箱 (smtp.126.com)
- Outlook (smtp-mail.outlook.com)
- 其他支持SMTP协议的邮箱服务
